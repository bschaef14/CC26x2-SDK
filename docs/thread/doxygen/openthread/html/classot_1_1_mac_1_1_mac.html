§ SetTemporaryChannel()

otError SetTemporaryChannel ( uint8_t  aChannel)

This method sets the temporary IEEE 802.15.4 radio channel.

This method allows user to temporarily change the radio channel and use a different channel (during receive) instead of the PAN channel (from SetPanChannel()). A call to ClearTemporaryChannel() would clear the temporary channel and adopt the PAN channel again. The SetTemporaryChannel() can be used multiple times in row (before a call to ClearTemporaryChannel()) to change the temporary channel.

Parameters
[in]aChannelA IEEE 802.15.4 channel.
Return values
OT_ERROR_NONESuccessfully set the temporary channel
OT_ERROR_INVALID_ARGSThe aChannel is not in the supported channel mask.

§ RecordFrameTransmitStatus()

void RecordFrameTransmitStatus ( const TxFrame &  aFrame,
const RxFrame *  aAckFrame,
otError  aError,
uint8_t  aRetryCount,
bool  aWillRetx 
)

This method records the status of a frame transmission attempt, updating MAC counters.

Unlike HandleTransmitDone which is called after all transmission attempts of frame to indicate final status of a frame transmission request, this method is invoked on all frame transmission attempts.

Parameters
[in]aFrameThe transmitted frame.
[in]aAckFrameA pointer to the ACK frame, or NULL if no ACK was received.
[in]aErrorOT_ERROR_NONE when the frame was transmitted successfully, OT_ERROR_NO_ACK when the frame was transmitted but no ACK was received, OT_ERROR_CHANNEL_ACCESS_FAILURE tx failed due to activity on the channel, OT_ERROR_ABORT when transmission was aborted for other reasons.
