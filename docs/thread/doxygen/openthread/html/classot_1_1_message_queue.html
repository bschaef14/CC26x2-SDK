Detailed Description

This class implements a message queue.

Member Enumeration Documentation

§ QueuePosition

enum QueuePosition

This enumeration represents a position (head or tail) in the queue. This is used to specify where a new message should be added in the queue.

Enumerator
kQueuePositionHead 

Indicates the head (front) of the list.

kQueuePositionTail 

Indicates the tail (end) of the list.
