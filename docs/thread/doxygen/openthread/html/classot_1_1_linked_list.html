Detailed Description

template<typename Type>
class ot::LinkedList< Type >

This template class represents a singly linked list.

The template type Type should provide GetNext() and SetNext() methods (which can be realized by Type inheriting from LinkedListEntry<Type> class).

This method searches within the linked list to find an entry and if found returns a pointer to previous entry.

Parameters
[in]aEntryA reference to an entry to find.
