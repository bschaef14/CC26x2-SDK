§ otPlatUartFlush()

otError otPlatUartFlush ( void  )

Flush the outgoing transmit buffer and wait for the data to be sent. This is called when the CLI UART interface has a full buffer but still wishes to send more data.

Return values
OT_ERROR_NONEFlush succeeded, we can proceed to write more data to the buffer.
OT_ERROR_NOT_IMPLEMENTEDDriver does not support synchronous flush.
OT_ERROR_INVALID_STATEDriver has no data to flush.
