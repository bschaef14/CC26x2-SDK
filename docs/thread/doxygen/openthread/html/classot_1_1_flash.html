This method fetches the value identified by aKey.

Parameters
[in]aKeyThe key associated with the requested value.
[in]aIndexThe index of the specific item to get.
[out]aValueA pointer to where the value of the setting should be written. May be NULL if just testing for the presence or length of a key.
[in,out]aValueLengthA pointer to the length of the value. When called, this should point to an integer containing the maximum bytes that can be written to aValue. At return, the actual length of the setting is written. May be NULL if performing a presence check.
Return values
OT_ERROR_NONEThe value was fetched successfully.
OT_ERROR_NOT_FOUNDThe key was not found.
