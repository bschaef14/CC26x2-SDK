This function parses the command line.

Note: this method may change the input aString, it will put a '\0' by the end of each argument, and aArgs will point to the arguments in the input aString. Backslash ('\') can be used to escape separators (' ', '', '', '
') and the backslash itself.

Parameters
[in]aStringA NULL-terminated input string.
[out]aArgsLengthThe argument counter of the command line.
[out]aArgsThe argument vector of the command line.
[in]aArgsLengthMaxThe maximum argument counter.
