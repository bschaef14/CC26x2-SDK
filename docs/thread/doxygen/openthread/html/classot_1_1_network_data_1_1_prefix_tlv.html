§ CalculateSize()

static uint16_t CalculateSize ( uint8_t  aPrefixLength)
inlinestatic

This static method calculates the total size (number of bytes) of a Prefix TLV with a given Prefix Length value.

Note that the returned size does include the Type and Length fields in the TLV, but does not account for any sub TLVs of the Prefix TLV.

Parameters
[in]aPrefixLengthA Prefix Length in bits.
