Member Enumeration Documentation

§ Mode

enum Mode

This enumeration defines the modes of operation for the TrickleTimer.

Enumerator
kModeNormal 

Runs the normal trickle logic (as per RFC6206).

kModePlainTimer 

Runs a plain timer with random interval selected between min/max intervals.

kModeMPL 

Runs the trickle logic modified for MPL.
