§ ReadTlv()

otError ReadTlv ( const Message &  aMessage,
uint8_t  aType,
void *  aValue,
uint8_t  aLength 
)
static

This static method searches for a TLV with a given type in a message, ensures its length is same or larger than an expected minimum value, and then reads its value into a given buffer.

If the TLV length is smaller than the minimum length aLength, the TLV is considered invalid. In this case, this method returns OT_ERROR_PARSE and the aValue buffer is not updated.

If the TLV is length is larger than aLength, the TLV is considered valid, but only the first bytes of the value are read and copied into the aValue buffer.

Parameters
[in]aMessageA reference to the message.
[in]aTypeThe TLV type to search for.
[out]aValueA buffer to output the value (must contain at least aLength bytes).
[in]aLengthThe expected (minimum) length of the TLV value.
Return values
OT_ERROR_NONEThe TLV was found and read successfully. is updated.
OT_ERROR_NOT_FOUNDCould not find the TLV with Type aType.
OT_ERROR_PARSETLV was found but it was not well-formed and could not be parsed.

References Tlv::GetValueOffset(), OT_ERROR_PARSE, Message::Read(), SuccessOrExit, and VerifyOrExit.
