§ SendFrame()

otError SendFrame ( const uint8_t *  aFrame,
uint16_t  aLength 
)

This method encodes and sends a spinel frame to Radio Co-processor (RCP) over the socket.

This is blocking call, i.e., if the socket is not writable, this method waits for it to become writable for up to kMaxWaitTime interval.

Parameters
[in]aFrameA pointer to buffer containing the spinel frame to send.
[in]aLengthThe length (number of bytes) in the frame.
Return values
OT_ERROR_NONESuccessfully encoded and sent the spinel frame.
OT_ERROR_NO_BUFSInsufficient buffer space available to encode the frame.
OT_ERROR_FAILEDFailed to send due to socket not becoming writable within kMaxWaitTime.

References Encoder::BeginFrame(), Encoder::Encode(), Encoder::EndFrame(), FrameBuffer< kSize >::GetFrame(), FrameBuffer< kSize >::GetLength(), OT_ERROR_NONE, SuccessOrExit, and HdlcInterface::Write().

If a full HDLC frame is decoded while reading data, this method invokes the HandleReceivedFrame() (on the aCallback object from constructor) to pass the received frame to be processed.
