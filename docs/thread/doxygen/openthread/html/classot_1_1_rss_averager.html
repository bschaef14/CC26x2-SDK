Detailed Description

This class implements a Received Signal Strength (RSS) averager.

The average is maintained using an adaptive exponentially weighted moving filter.

§ Add()

otError Add ( int8_t  aRss)

This method adds a received signal strength (RSS) value to the average.

If aRss is OT_RADIO_RSSI_INVALID, it is ignored and error status OT_ERROR_INVALID_ARGS is returned. The value of RSS is capped at 0dBm (i.e., for any given RSS value higher than 0dBm, 0dBm is used instead).

Parameters
[in]aRssReceived signal strength value (in dBm) to be added to the average.
Return values
OT_ERROR_NONENew RSS value added to average successfully.
OT_ERROR_INVALID_ARGSValue of aRss is OT_RADIO_RSSI_INVALID.
