Initially, a radio is in the Disabled state.

The following are valid radio state transitions for the platform

+----------+ Enable() +-------+ Receive() +---------+ Transmit() +----------+
| |----------->| |------------->| |-------------->| |
| Disabled | | Sleep | | Receive | | Transmit |
