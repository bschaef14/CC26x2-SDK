Running the Example Applications¶

This section provides an overview of the Z-Stack out-of-box example applications and instructions on how to run them. The examples are out-of-box compliant with the ZCL v7 Specification but will further Product Certification will be required.

Note

Most examples have a coordinator, end device and router application. Green Power Device examples complement the Sink coordinator/router examples. The Network Processor is standalone. Generic App provides the template code for application development with minimal overhead (i.e. no ZCL support included) whereas Sample App incorporates the SysConfig Application Builder to help configure the required Zigbee application device type, clusters/attributes, commands, and command callback APIs.

Note

All the sample applications are supported for the CC2652[R/P/RB] and CC1352[R/P] (2.4 GHz) platforms. If a different platform is intended to be used, just replace the board name in the path to find the corresponding README document.(e.g. “examples/rtos/CC1352R1_LAUNCHXL/zstack/zc_doorlock/README.html” if CC1352R1 platform is intended).
